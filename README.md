Theme inspired by default Eclipse theme.

![Xcode] [xcode]
![Eclipse] [eclipse]

To install the theme:

1. Copy the Eclipse theme.dvtcolortheme to ~/Library/Developer/Xcode/UserData/FontAndColorThemes/.
2. Restart Xcode 4.
